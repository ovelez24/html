Numeros big int

Las variables de javascript Bigint se utilizan para almacenar valores enteros grandes
que son demasiado grandes para ser representados por un Javascript normal Number

En JavaScript, todos los números se almacenan en un formato de punto flotante de 64 bits (estándar IEEE 754).
Con este estándar, los enteros grandes no se pueden representar exactamente y se redondearán.
Debido a esto, JavaScript solo puede representar números enteros de manera segura: Hasta 9007199254740991 +(2 53 -1) y Hasta -9007199254740991 -(2 53 -1). Los valores enteros fuera de este rango pierden precisión

Como crear un BigInt

Para crear un BigInt, agregue n al final de un numero entero o llame a BigInt():

BigInt es el segundo tipo de dato numerico en JavaScript despues de Number.

Operadores BigInt

Los operadores que se pueden usar en javaScript Number tambien se pueden usar en un archivo BigInt.
